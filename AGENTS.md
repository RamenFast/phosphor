# AGENTS.md — Phosphor

Ben's Rust CRT oscilloscope, a station node — probe/ctl/tap CLI surface, themes, packaging.

## SOP: feedback ledger (binding)

On opening this directory in a session, track every atomic claim, ask,
correction, bugfix, feature request, and UI change Ben gives into
`FEEDBACK.md` as you work. Rules live at the top of that file:

- One dated line per atomic item, newest at the bottom.
- Duplicates are never re-added: keep/tighten the existing line and prefix a
  repeat counter (`- 2x ...`), updating the date. Repeats are signal.
- `FEEDBACK.md` requires Ben's permission to delete.
- Take no autonomous action from the ledger. Ben mines it and updates this
  file himself.
